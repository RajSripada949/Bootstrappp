Intelligent Invoice Processing System Using Tesseract OCR

An invoice is a commercial document issued by a seller to a buyer, providing a detailed break down of products with the prices. The traditional paper based invoice invoice processing is often time consuming, errorprone and costly, leading to details in payments & poor supplier relationship. Existing & Proposed Models: In this project we help them by retrieving the required data from the Invoice provided for analysis. In the Existing project the output text is extracted only in text fromat, JSON was not implemented and the accracy obtained was low. In our proposed model we are using JSON format, which is very efficient to use the data in other applications. Software Implementation: In this project, we will extract the text from the invoice using Tesseract OCR engine. Tesseract OCR engine is a "Open Source Software Library" developed by google. It is designed to recognize and extract text from image or scanned documents. Implemented Algorithm: This project uses Tesseract, which internally uses LSTM algorithm, this algorithm extracts the data from the image. The activation functions used are softMax and Tanh Process: It uses ML Algorithms to identify text in various languages and convert it into a machine readable format. In our project we aim to provide the O/P text in the structured format i.e JSON format. Conclusion: The project utilizing Tesseract OCR outomates the extraction of relevent information from invoices, with preprocessing, OCR algorithms. The data validation module ensures accuracy and integrity and improves efficiency in Invoice processing. THANKQ--
